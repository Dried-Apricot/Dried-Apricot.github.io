# Algorithm
1. `input = sys.stdin.readline`과 `input = sys.stdin.readline()`의 차이

전자를 입력하면 input 은 method 가 된다.
`input()`이 `sys.stdin.readline()`과 동일한 의미가 된다.


후자를 입력하면 input이라는 이름의 변수에 readline()을 하겠다는 의미이다.
